# Frontend (React + Vite)

## Prérequis

- Node.js + npm
- Le backend démarré sur `http://localhost:8080`

## Lancer en local

```sh
npm i
npm run dev
```

Le frontend tourne par défaut sur `http://localhost:5173` et proxy automatiquement les appels `/api/*` vers le backend (`http://localhost:8080`).
